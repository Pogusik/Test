Обновление v1.1

  • Новая система категорий
    Добавлено удобное разделение предметов по категориям и подкатегориям в инвентаре для быстрой навигации
  • Визуализация предметов
    Теперь отображаются иконки предметов, соответствующие их игровым моделям
  • Улучшенная палитра цветов
    Добавлены подсказки с названиями и кодами цветов при наведении. Некорректные цвета для SCP:SL были заменены
  • Исправления ошибок
    - Устранена проблема с подсчетом символов в описаниях
    - Оптимизирована работа с цветовыми кодами
    - Мелкие исправления интерфейса

Hot Fix v1.1.1

  • Улучшения
    - Добавлен счётчик предметов в каждой категории
  • Исправления ошибок
    - Исправлена ошибка, которая могла ломать списки категорий
    - Исправлено отображение кастомных предметов (LightRP теперь корректно отображается)
    - Мелкие исправления интерфейса
